# perl5-minilla README

This is the README for the extension "perl5-minilla". 

## Description

perl5-minilla extension provides some Minilla features to VSCode seamlessly.

## Features

* install Minilla
* minil new 
* minil build
* minil test (ctrl + shift + j)
* minil release

## Requirements

* Perl 5.8.8 or later
* cpanm
